# V5 Roadmap — Honest History (written 2026-07-18)

**Theme:** the forward track record (day ~40) is honest but young. V5 buys the
one thing money can buy here — survivorship-free history — and uses it for an
HONEST backtest-to-today of the exact lane mandates, third-party-verified,
so we can compare performance NOW instead of waiting 24 months. Plus the NN
trial that clean data finally makes legitimate.

## Non-negotiables (unchanged from CANON)

1. **The backtest is a direction-check, not the track record.** Historical
   replay results are displayed on a SEPARATE methodology/history view and
   written to research docs — **never into `paper_nav`** (CANON §5, the
   write-path is sacred). A good backtest number is never reported as "we
   beat the market."
2. **Purged CV + DSR/PBO deflation** on anything where a parameter was
   chosen. Even on clean data, picking the best of N variants is overfitting.
3. **Six forward clocks is the cap.** Everything in V5 is offline research
   until a trial is explicitly pre-registered (attended, Murat-approved).
4. **EODHD is offline-validation-only.** The token stays local / research
   scripts; it does not become a live serving dependency on Railway.
5. Every gate has a pre-committed pass bar written BEFORE the data is seen.
