**What I did:** Rewrote `project_portfolio` to use the same GARCH-enhanced MC:
1. **GARCH fit** on portfolio returns → conditional vol + Student-t df

3. **Log-space simulation** `exp(log_return)` → impossible to get negative prices
4. **Ito correction** `drift = log(1+r) - 0.5σ²` → correct expected returns
5. **OU volatility dynamics** with adaptive Ito correction → time-varying vol
