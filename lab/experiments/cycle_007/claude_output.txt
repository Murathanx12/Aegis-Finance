## Cycle 7 Results

### Fix 1: Ito Correction in `run_monte_carlo` (monte_carlo.py)
**Bug**: `historical_mu = np.log(1 + base_annual_return)` is the continuous compounding rate (GBM `mu`), but `simulate_paths` treats it as the log-return drift (`mu - 0.5*sigma^2`). The missing Ito correction biased SP500 MC arithmetic returns **+1.28%/yr** above target.

**Fix**: `historical_mu = np.log(1 + base_annual_return) - 0.5 * historical_sigma**2`

|--------|--------|-------|
| SP500 MC arithmetic annual return | 7.36% | **6.00%** (matches institutional consensus) |
| Overshoot vs target | +1.36%/yr | **0.00%** |
