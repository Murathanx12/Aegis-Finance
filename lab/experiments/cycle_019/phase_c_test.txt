### What the narrow excepts catch vs what they let through

**Caught** (graceful degradation — feature unavailable, fallback to `None`):
- `ImportError` — crash model or features module not installed
- `FileNotFoundError` — model .pkl not trained yet
- `ValueError` / `KeyError` — data shape mismatch, missing columns
- `TypeError` — wrong argument types in validator

**No longer caught** (will surface as 500 errors to diagnose real bugs):
- `AttributeError` — calling wrong method names (the old sector.py bug)
- `IndexError` — empty DataFrames slipping through
- `RuntimeError` — model corruption
